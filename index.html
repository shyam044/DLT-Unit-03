2) Anatomy of Neural Network

Layer:

  • Layer is the Modular units that process the data
  • Layers can be stateless or stateful
  • Layer types:
    • Convolutional Layers
    • Recurrent Layers

Model(Network):

  • It is a DAG of layers that maps inputs to outputs
  • They are built using:
    • Sequential API
    • Functional API
  • Sequential API is used for simple models
  • Functional API is used for complex models

Input and Target:

  • They must be properly shaped before feeding into the Network

Loss Functions:

  • Loss function is the difference between the predicted value and the actual value

Optimizer:

  • Optimizer updates the weight based on the loss functions

5) Binary classification(classifying movie reviews)

  • Binary classification for movie reviews involves classifying each movie review as positive or negative.

Input:

  • Use the IMDb dataset.
  • It contains 25,000 training samples and 25,000 testing samples.

Pre-Processing:

  • Transform all text into lowercase and remove all special characters.
  • Convert the sentences into a list of tokens.

Feature Extraction:

  • Use Keras's Embedding Layers to convert tokens into vectors.
  • Example:
    • a = 100
    • b = 10
    • Embedding(a, b)

Classification:

  • After the Embedding Layer, add a Flatten Layer to reduce the dimensions.
  • Add one or more Dense Layers.
  • Add a final Dense Layer for binary output.
  • The model classifies each movie review as positive or negative.
